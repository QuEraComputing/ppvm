# generated/noise_sweeps/

Per-channel parameter sweeps. Generated by:

```bash
cd crates/ppvm-stim/tests/regen-stim
uv run regen-stim noise-sweeps
```

Each fixture has uniform shape: `H`-prepare → apply-channel-N-times → `M` all.
Sweeping the probability axis gives the regen-time Stim cross-check enough
breadth to catch slope/sign bugs on each channel.

Channels covered: `DEPOLARIZE1`, `DEPOLARIZE2`, `PAULI_CHANNEL_1`,
`PAULI_CHANNEL_2`, `X_ERROR`, `Y_ERROR`, `Z_ERROR`, plus measurement-readout
noise via `M(p)` / `MR(p)`.

Fixtures are distribution-mode at `num_shots=4096`. Statistical signal needs
many shots; each circuit is tiny so the wall time is acceptable.

Loss channels (`I_ERROR[loss]`) are absent — Stim has no oracle for them.
See spec Non-Goals.
