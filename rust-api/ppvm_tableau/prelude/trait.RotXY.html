Expand description

Rotation about an axis in the x/y plane: R(axis_angle, θ) = exp(-i θ/2 · (cos(axis_angle)·X + sin(axis_angle)·Y)).

The in-plane axis is X rotated about Z by axis_angle, so R(axis_angle, θ) = RZ(axis_angle)·RX(θ)·RZ(−axis_angle).
