Expand description

+= semantics for an ACMap: insert a new entry or accumulate into the existing one with the same key.

Required Methods§

fn add_assign(&mut self, key: W, value: V)

Add value into the entry at key, creating the entry if absent.

fn map_add_assign<F>(&self, dest: &mut Self, f: F)
where F: Fn(&W, &V) -> (W, V) + Sync + Send,

For every entry, compute f(key, value) and add the result into dest with add_assign.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
