XOR-combinable hash of is_lost plus every row’s phase, formed as the XOR of per-row contributions (the phase/loss half of [fingerprint]). Being XOR-combinable lets a branch inherit its parent’s value and update only the rows it changed — a sign flip XORs [sign_mask], a loss XORs [loss_mask].
