Hash of the word (Pauli content) of every row, in order. This is the expensive component (each word is several machine words wide) and is invariant under X/Y/Z and is_lost flips, so a branch inherits it from its parent unchanged. NOTE: this inheritance is only valid right now (loss + depolarize channels) but may need re-evaluation in the future when more gates are added
