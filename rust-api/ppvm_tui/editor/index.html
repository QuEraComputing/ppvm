ppvm_tui

Module editor 

Source
Expand description

A readline-style single-line editor with command history. Owns the command buffer, edit cursor, and history; knows nothing about the PPVM or the debugger. The host handles submit (Enter) and quit keys, then delegates the remaining editing keys here via LineEditor::handle_key.
