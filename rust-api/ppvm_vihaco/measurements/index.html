Union of the two effect types a circuit instruction can produce. The plan’s structural note (Task 6) calls for broadening the #[component(..., effect = ...)] annotation on Circuit and the executors to a union so a single Trace (or Measure) instruction can fan out to the right observer.
MeasurementOutcome
Measurement results are represented as an integer enum: 0: state |0> 1: state |1> 2: qubit has been lost prior to measurement In bytecode, this is represented as a u32 integer, which is simpler than e.g. two boolean values and matches semantics elsewhere
