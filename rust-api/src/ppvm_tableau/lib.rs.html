1//! Generalized stabilizer-tableau simulator built on top of `ppvm-runtime`.
2//!
3//! Provides a forward-evolving state representation using the stabilizer
4//! formalism, extended to non-Clifford gates by tracking a sparse vector
5//! of coefficients indexed over bitstrings. The two top-level types are
6//! [`data::Tableau`] (pure Clifford) and [`data::GeneralizedTableau`]
7//! (Clifford + non-Clifford with sparse coefficient tracking).
8//!
9//! # Quick example
10//!
11//! Prepare a Bell pair and verify the two measurements are perfectly
12//! correlated:
13//!
14//! ```
15//! use ppvm_tableau::prelude::*;
16//! use ppvm_runtime::config::fxhash::ByteF64;
17//!
18//! let mut tab: GeneralizedTableau<ByteF64<1>> =
19//!     GeneralizedTableau::new_with_seed(2, 1e-12, 0);
20//! tab.h(0);
21//! tab.cnot(0, 1);
22//!
23//! let r0 = LossyMeasure::measure(&mut tab, 0);
24//! let r1 = LossyMeasure::measure(&mut tab, 1);
25//! assert_eq!(r0, r1);
26//! ```
27
