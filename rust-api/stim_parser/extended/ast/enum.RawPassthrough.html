Expand description

Subset of RawInstruction that passes through the extended-dialect interpreter unchanged. Excludes MPad and Repeat, which are always lowered to the typed ExtendedInstruction::MPad / ExtendedInstruction::Repeat variants — so by construction ExtendedInstruction::Raw(_) can never wrap one of them.
