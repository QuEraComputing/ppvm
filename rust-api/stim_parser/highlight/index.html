Module highlight 

Source
Expand description

Syntax highlighter for Stim source text — a public, renderer-agnostic companion to the canonical printer.

highlight_html and highlight_ansi take any Stim string (typically the canonical output of StimPrint::to_stim, but raw input works too) and wrap its tokens for display. The lexer is line-aware — the first identifier on a line is the instruction mnemonic (or REPEAT) — so it never has to be a full parser.
