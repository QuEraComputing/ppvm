Expand description

The parse trait wired up by #[derive(Parse)], re-exported so downstream crates can call CircuitInstruction::parser without depending on vihaco-parser-core directly to bring the trait into scope. A parser whose input is &'src str (char stream) and whose error type is Simple<char>.

The lifetime 'src is the input lifetime. Output type Self is owned and does not borrow from the input.
